# CyberSec Quiz

An interactive cybersecurity quiz built with pure HTML, CSS and JavaScript.

## Live Demo
https://alishbadua22-bot.github.io/cybersec-quiz

## Features
- 15 questions on cybersecurity topics
- Instant correct/wrong feedback
- Detailed explanation after every answer
- Score tracker
- Play again button

## Topics Covered
- Fundamentals (CIA Triad, VPN)
- Cryptography (Hashing, RSA, AES)
- Networking (HTTP, SSH, HTTPS)
- Attacks (DoS, Phishing, SQL Injection)
- Linux (chmod, find command)
- Tools (Nmap, Wireshark)

## Built With
- HTML
- CSS
- JavaScript
